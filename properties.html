So far, I have learnt about GitHub, HTML and CSS. I'm currently learning CSS which means Cascading Style Sheets and it plays a very important role in web development as it's responsible for styling. Without CSS, our web pages would be bland and devoid of color. There are three ways to add CSS to your HTML code, Inline CSS, Internal CSS styling, and External CSS styling. CSS has various selectors including class selector, universal selector, id selector, attribute selector etc.

Padding: Padding is the space between an element's border and the element's content. Adding padding to your element adds space between the text and the border of the element.
Margin: Margin is the space around an element's border. It is used to define the space around elements. Adding padding to your element clears an area around the element.
Border: Border is used to set the border of an element. It's the perimeter of the element and it can either be invisible or set to a thick line. It could be dotted or broken too.
